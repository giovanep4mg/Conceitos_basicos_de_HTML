Conceitos Básicos de HTML


O que é HTML ?


HTML, ou HyperText Markup Language, é a linguagem padrão para criar páginas web.
Vamos abordar alguns conceitos básicos:


Tag


Tags são elementos fundamentais no HTML e são usadas para estruturar o conteúdo da página. Elas são representadas entre colchetes angulares. Por exemplo: < tag > conteúdo < / tag > . As tags geralmente têm uma versão de abertura e uma versão de fechamento.

Elementos:


Um elemento HTML é constituído por uma tag de abertura, conteúdo e uma tag de fechamento.
Por exemplo:
< p >Isso é um parágrafo

é um elemento de parágrafo.


Para saber mais sobre Elementos HTML
Clique aqui.

Atributos:


As tags podem ter atributos que fornecem informações adicionais sobre como o elemento deve ser exibido ou se comportar. Atributos são incluídos dentro da tag de abertura.

Por exemplo:
Usamos a tag "a" : a
Adicionamos o atributo "href" : href=""
Dentro dela colocamos o endereço : https://www.exemplo.com

Quando o usuário clicar no link, será redirecionado para o endereço do link.

Estrutura Básica:


Um documento HTML básico começa com < !DOCTYPE html > e tem elementos como < html >, < head >, < title >, e < body >.

Elementos Semânticos:


Elementos semânticos fornecem significado ao conteúdo, como < header > , < nav >, < article >, < section >, < footer > .
Eles ajudam a estruturar o conteúdo de maneira mais significativa.


Observação final:


Estes são apenas alguns dos conceitos básicos do HTML. À medida que você avança, pode explorar mais tags, atributos e conceitos mais avançados para criar páginas web mais complexas e estruturadas.

Exercício 01:


Exercício simples para praticar o básico do HTML:

Exercício: Criando uma Página Básica


Crie um novo documento HTML.

Adicione a estrutura básica do HTML, incluindo as tags < !DOCTYPE html >, < html >, < head >, < title > , e < body > .

Defina um título para a sua página dentro da tag < title >.

Adicione um cabeçalho ( < h1 > ) com o texto "Minha Primeira Página HTML" no corpo da página.

Abaixo do cabeçalho, crie dois parágrafos ( < p > ) com algum texto de sua escolha.

Insira um link ( < a > ) que aponte para a página inicial do Google (https://www.google.com). Use o texto "Ir para o Google" como o conteúdo do link.

Adicione uma lista ordenada ( < ol > ) com pelo menos três itens.

Insira uma imagem ( < img > ) em sua página. Você pode usar a URL de uma imagem online ou fazer o download de uma imagem para o seu computador e referenciá-la localmente. Não se esqueça de adicionar o atributo alt para uma descrição da imagem.

Finalize sua página com uma breve despedida ou assinatura no final.

Salve o arquivo com uma extensão .html e abra-o em um navegador para visualizar sua página.

Observação:
Este exercício ajudará a praticar a criação de estrutura HTML básica, utilização de tags, e a compreensão de como os diferentes elementos se relacionam.
